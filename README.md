### Installation

Clone the repo

```
git clone https://github.com/WahabDev12/Gradients-Ninja.git
````
Go into project directory

```
cd Gradients-Ninja
```

Install packages and dependencies

```
// Do this in the root folder

npm install
```

Run application

```
npm start
```

View app on [localhost://3000](localhost://3000)
